# mmorpg学习项目

MMO-Client：客户端的源码。

MMO-SERVER：服务端源码。

MMO-Client-Release：客户端的游玩版本，可以连接来本人的服务器进行联机游玩(关闭了)。

Tools：里面有proto工具和excel工具、还有构建mysql数据库的sql文件

MMORPG.md:是本人在学习项目的时候所做笔记。
